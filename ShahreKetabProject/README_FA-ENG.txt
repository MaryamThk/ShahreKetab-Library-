- گزارش موجودی‌های بحرانی
- جستجو و گزارش‌گیری

# Shahr-e Ketab Chain Store Management System

Final project for the Database Lab course. A desktop application for managing a chain of bookstores, built with Python and Tkinter, using SQL Server as the database.

## Features

- Store management
- Book management
- Inventory management
- New sales registration
- Debtor customers tracking
- Critical inventory reports
- Search and reporting

## Technologies

- Python
- Tkinter (GUI)
- SQL Server (database)
- PyODBC (database connection)

## Prerequisites

- Python 3.x
- SQL Server with the `ShahreKetabDB` database
- ODBC Driver 17 for SQL Server
- pyodbc library:
